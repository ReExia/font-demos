## Angular组件

1.template-lang : 模板语法 插值表达式

2.template-ref : 模板语法 局部变量

3.value-bind : 属性值绑定

4.event-bind : 事件绑定

5.each-bind : 双向绑定

6.ng-If : 指令ngIf的使用

7.use-ng-for : ngFor的使用

8.use-ng-class : ngClass使用

9.use-pipe : 内置管道及自定义管道的使用

## 安装和运行
    
```
npm config set registry https://registry.npm.taobao.org
```

```
npm install
```

```
npm start
```
